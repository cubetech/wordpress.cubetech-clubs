== Description ==

cubetech Clubs - a nice plugin for managing and listing clubs in a table, also with jQuery category filtering

Plugin features:

* Custom post type
* Short code
* Short code button in TinyMCE
* Uses builtin jQuery
* Filtering with jQuery by groups (can be disabled) 

== Installation ==                                                                                                                                                
                                                                                                                                                                  
This section describes how to install the plugin and get it working.                                                                                              
                                                                                                                                                                  
1. Upload plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress                                                                                                    
1. Place the shortcode in your content
